## vscode

- [`prettier-vscode` plugin](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- _preferences_ > _Open Settings (JSON)_ > paste this:

```json
// Set the default
"editor.formatOnSave": false,
// Enable per-language
"[javascript]": {
    "editor.formatOnSave": true
}
```
